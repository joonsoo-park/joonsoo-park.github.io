# 희소 로그 패키징
- #### 희소 로그 가이드
- [[희소 로그 반영 가이드]]
-
- 희소 로그는 패키지로 관리
- 패키지 생성 시, 기본으로는 빠져있고 선택적으로 포함될 수 있어야 한다
-
- ### 희소 로그 포함해 패키지 생성하는 방법
- `version/optionsl.env` 파일을 추가해 희소 로그 포함 여부를 관리합니다
- ```env
  # 선택 모듈 포함 여부 토글
  # 버전은 version/image/<module> 에서 항상 관리하고, 패키지 포함 여부만 여기서 제어한다.
  # true 인 경우에만 해당 모듈이 패키지에 포함된다. (기본값: false)
  
  # sparselog (로그 희소 패턴 분석) 포함 여부
  INCLUDE_SPARSELOG=true
  ```
- `INCLUDE_SPARSELOG` 값을 `true` 로 설정하면 태그 생성 시 희소 로그가 포함됩니다
